# Mystical Animal Counseling

## Description
신비로운 동물 상담소

## Commit Rule

- 🛠refactor : 새로운 기능이 아닌 코드를 재정비 하였을 때 (변수 이름 바꾸기)
- 🐛fix : 버그 수정
- 🧸feat : 새로운 기능 추가
- 🧀chore : 빌드 업무 수정, 패키지 매니저 수정
- 🌱docs : 문서 변경
- 🍰test : 누락된 테스트 추가, 생산적인 코드 생성은 없음

## Spec

- MVVM Architecture Pattern
- Databinding
- AAC(Android Architecture Component)
